# ChurnSense.AI 

AI-powered Streamlit app that allows users to analyze telecom churn datasets using natural language questions.

## Features

- Natural language dataset querying
- Automatic Python (pandas) code generation
- Interactive visualizations
- Chat-based interface
- Streamlit dark gaming theme UI
- Excel dataset support

## Example Questions

- What percentage of customers churned?
- Which contract type has the highest churn?
- Average monthly charges of churned customers
- Plot churn distribution by internet service

## Run locally

pip install -r requirements.txt

streamlit run app.py
